# 에코퀘스트

1. **기획 배경 및 목적**
   
   - 일상에서 만나는 생물에 대한 호기심을 해소하고, 추가적인 생태적 관심을 유도.
   
   - 유저가 흥미를 가질 수 있는 생물 도감 수집 게임화 서비스를 넘어, 생태학적으로 유용한 실시간에 가까운 데이터를 생성하는 데이터 플랫폼 구축.

2. **타겟 사용자**
   
   - **사용자 (데이터 생산자)**
     생물에 대한 호기심이 있고 수집형 게임을 좋아하는 사람에게 수집형 퀘스트를 통한 게임형 서비스를 제공함으로써 호기심을 지식 습득으로 연결하게 해주는 경험을 제공한다.
   
   - **생태학자/기관 (데이터 수요자)**
     기존에는 도시권 생태계 데이터 수집에 막대한 시간과 비용이 발생했지만, 에코퀘스트로 최신의 고품질 데이터를 적은 비용으로 얻을 수 있게 된다.

3. **핵심 기능**
   
   1. 생물 렌즈 및 도감화
      
      - 사진 업로드 시 생물을 인식하고 사진 메타데이터와 함께 ‘도감’에 저장.
      
      - 이를 통해 사용자에게는 수집의 경험을, 생태학자에게는 좋은 데이터를 얻을 수 있음.
   
   2. 가상 테라리움
      
      - 수집한 생물을 가상 테라리움에 배치하고 생물들과 상호작용하여 여러 생태적 특징을 학습.
   
   3. 퀘스트
      
      - 수집욕과 작은 성취 목표를 제공하기 위해 퀘스트 제공.
      
      - 이 퀘스트를 통해 테라리움을 꾸밀 때 사용할 수 있는 보상을 받을 수 있음.
      
      - 더불어 날씨나 시공간 정보를 바탕으로 생태계의 동적인 모습을 보여줄 수 있는 퀘스트를 추가로 제공함.
      
      - 생태학자들이 특정 데이터를 수집하고 싶을 때, 이를 퀘스트 형태로 데이터를 수집하도록 할 수 있게 데이터 수집 도구로 사용.
   
   4. 검증 미니게임
      
      - 사진 업로드에 거짓된 사진이나 시스템의 잘못된 판단을 검증하기 위해 다른 사용자들에게 업로드된 사진을 퀴즈 형식으로 생물 맞추기 미니게임을 진행. 이를 통해 사진을 상호 검증해 데이터의 신뢰성을 높임.
   
   5. 생태계 보호 시스템
      
      - 자연보호구역에 살거나 관찰이 생태계 훼손을 일으킬 수 있는 생물의 경우 수집 보상 및 도감 수집 목표에서 제외하는 동시에 퀘스트 보상으로 얻을 수 있도록 하여 보호종을 보호하는 동시에 수집과 정보 획득의 길을 열어놓음.

4. **기존 서비스와의 차별점**
   
   - 현 기획안과 비슷한 서비스는 iNaturalist와 네이처링이 있다.
   
   - 이들과 에코퀘스트의 차이는 게임화 시스템이다. 타 서비스는 아마추어 생태학자들의 기여를 받아 만드는 생태 지도 서비스의 형태이지만, 에코퀘스트는 수집의 요소를 넣어 생태에 관심있는 모든 사람들로 범위를 확장하고 사용자가 지속적으로 기여할 수 있도록 하였다.
   
   - 또한 네이처링의 한계인 잘못된 데이터 문제를 에코퀘스트는 캡차(CAPTCHA) 형식의 미니게임을 통해 AI에만 의존하지 않는 시스템을 만들어 기존 서비스를 개선하는 형태로 기획하였다.
